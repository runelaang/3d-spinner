3d-spinner and beyond: motion paths

ObjectMotionAnimation moves any mesh along a motion controller - a separate, swappable object that decides how the thing moves. Swap circleMotion, squareMotion, figureEightMotion, or wanderMotion (or write your own) and the same animation follows that path, nose along the tangent. Each card runs as an indeterminate spinner; the config used to build it is shown beneath.
